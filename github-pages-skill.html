FAQ

常见问题

需要安装 Git 吗？▼
不需要！本 Skill 完全通过 GitHub REST API 操作，用 Python 的 requests 库直接上传文件，零安装。
Token 权限需要哪些？▼
需要 repo 权限（完整仓库读写）。在 GitHub Settings → Developer settings → Personal access tokens 生成。
Pages 部署后多久生效？▼
通常 1-3 分钟。GitHub 需要构建和部署页面，可以在仓库 Settings → Pages 查看构建状态。
上传的文件大小有限制吗？▼
GitHub Contents API 单文件限制 100MB。对于 HTML/CSS/JS 页面来说完全够用。
和 ngrok 方案怎么配合？▼
ngrok 适合临时预览（即时访问，地址会变），GitHub Pages 适合正式发布（永久地址，几分钟生效）。开发阶段用 ngrok 快速验证，满意后用 GitHub Pages 正式部署。

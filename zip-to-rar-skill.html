概述

将文件或目录打包为 RAR 格式压缩包。转换流程：

  1. 7-Zip 将源文件/目录打包为 ZIP
  2. CloudConvert API 将 ZIP 在线转换为 RAR
  3. 下载 RAR 文件到本地

前置条件

系统依赖

  • 7-Zip：默认路径 C:\Program Files\7-Zip\7z.exe，用于本地打包 ZIP
  • Python 依赖：cloudconvert、requests

CloudConvert API Key

  • 需要用户提供 CloudConvert API Key
  • 免费注册地址：https://cloudconvert.com/register
  • API Key 获取：Dashboard → API → API Keys → Create API Key
  • 免费额度：每天 25 次转换，单文件最大 1GB

使用流程

1. 获取源文件

  • 如果用户通过企微发送了文件附件，先下载到本地临时目录
  • 如果用户指定了本地路径，直接使用

2. 执行转换

python scripts/zip_to_rar.py \
  --api-key <CLOUDCONVERT_API_KEY> \
  --source <源文件或目录路径> \
  --output-dir <输出目录> \
  --cleanup

参数说明

参数 必填 说明
--api-key 是 CloudConvert API Key
--source 是 要压缩的文件或目录
--output-dir 是 输出目录，默认使用 D:\WorkBuddyOutput\YYYY-MM\
--cleanup 推荐 转换完成后删除中间 ZIP 文件
--seven-zip 否 7-Zip 路径，默认 C:\Program Files\7-Zip\7z.exe

3. 输出结果

  • RAR 文件保存到 --output-dir 指定目录
  • 文件名与源文件同名，扩展名改为 .rar
  • 通过 deliver_attachments 将 RAR 文件发送给用户

注意事项

  • API Key 隐私：不要在日志或输出中打印完整的 API Key
  • 文件大小：CloudConvert 免费版单文件最大 1GB，超出需升级
  • 网络依赖：转换过程需要上传和下载，网速慢时可能耗时较长
  • 转换额度：每天 25 次，注意不要浪费在测试上
  • 7-Zip 不可用：如果系统没有 7-Zip，提示用户安装或改用 Python 的 zipfile 模块替代打包步骤
  • 中间文件：加上 --cleanup 参数自动清理中间 ZIP 文件
⚠️ 注意： config.env 文件中包含敏感凭证，请勿泄露。

故障排查

问题 原因 解决方案
7-Zip not found 7-Zip 未安装或路径不对 安装 7-Zip 或用 --seven-zip 指定路径
Unauthenticated API Key 无效或过期 重新生成 API Key
Task 'convert' failed 文件格式不支持或损坏 检查源文件是否正常
下载超时 网络问题 重试

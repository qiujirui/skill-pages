ZIP → RAR

基于 7-Zip + CloudConvert API，将任意文件或目录自动转换为 RAR 压缩包。零配置、全自动。

压缩 格式转换 RAR
📦 下载技能包 (zip-to-rar.zip)

1概述

ZIP → RAR Skill 是 WorkBuddy 的格式转换工具，核心解决 Windows 系统无法原生创建 RAR 文件的痛点。

由于 RAR 是 WinRAR 公司的专有闭源格式，7-Zip、Python 等开源工具只能解压，无法创建 RAR 文件。本 Skill 通过 7-Zip 本地打包 ZIP → CloudConvert API 云端转换 → 下载 RAR 的全自动流程解决此问题。

2核心痛点

❌ 专有格式限制
RAR 是闭源专有格式，开源工具无法创建 RAR 文件。
❌ 安装门槛高
WinRAR 安装需要管理员权限，远程操作时无法静默安装。
✅ 云端转换方案
CloudConvert 提供 REST API，支持 200+ 格式互转，免费额度每天 25 次，无需安装任何软件。

3使用流程

第 1 步：获取源文件

  • 用户通过企微发送文件附件，或指定本地路径

第 2 步：执行转换

python scripts/zip_to_rar.py \ --api-key <CLOUDCONVERT_API_KEY> \ --source <源文件或目录路径> \ --output-dir <输出目录> \ --cleanup

参数说明：

  • --api-key：CloudConvert API Key（必填）
  • --source：要压缩的文件或目录（必填）
  • --output-dir：输出目录，默认 D:\WorkBuddyOutput\YYYY-MM\
  • --cleanup：转换完成后删除中间 ZIP 文件
  • --seven-zip：7-Zip 路径，默认 C:\Program Files\7-Zip\7z.exe

第 3 步：输出结果

  • RAR 文件保存到指定输出目录
  • 文件名与源文件同名，扩展名改为 .rar
  • 通过 deliver_attachments 将 RAR 文件发送给用户

5CloudConvert API 调用链

1️⃣ 创建 Job

POST /v2/jobs — 创建包含 upload → convert → export 三个任务的 Job

2️⃣ 上传文件

PUT upload_url — 将 ZIP 文件二进制流上传到 CloudConvert 存储

3️⃣ 等待转换

GET /v2/jobs/{id}/wait — 轮询等待 convert 任务完成

4️⃣ 获取下载链接

从 export 任务结果中提取 RAR 文件的临时下载 URL

5️⃣ 下载 RAR

GET download_url — 下载 RAR 文件并保存到本地输出目录

6故障排查

UnauthenticatedAPI Key 无效或过期重新生成 API Key
Task 'convert' failed文件格式不支持或损坏检查源文件是否正常
下载超时网络问题重试

7注意事项

  • API Key 隐私：不要在日志或输出中打印完整的 API Key
  • 文件大小：CloudConvert 免费版单文件最大 1GB，超出需升级
  • 网络依赖：转换过程需要上传和下载，网速慢时可能耗时较长
  • 转换额度：每天 25 次，注意不要浪费在测试上
  • 中间文件：加上 --cleanup 参数自动清理中间 ZIP 文件

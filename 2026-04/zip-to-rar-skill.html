背景

为什么需要这个 Skill？

RAR 是专有格式，Windows 系统无法原生创建，只有 WinRAR 才能生成

🔒

专有格式限制

RAR 是 WinRAR 公司的闭源专有格式，7-Zip、Python 等开源工具只能解压，无法创建 RAR 文件

💻

安装门槛高

WinRAR 安装需要管理员权限，远程操作时无法静默安装，用户不在电脑前就无法完成

🌐

云端转换方案

CloudConvert 提供 REST API，支持 200+ 格式互转，免费额度每天 25 次，无需安装任何软件

API Deep Dive

CloudConvert API 调用链

5 步 API 交互，完成从上传到下载的全过程

1️⃣ 创建 Job

POST /v2/jobs — 创建包含 upload → convert → export 三个任务的 Job

2️⃣ 上传文件

PUT upload_url — 将 ZIP 文件二进制流上传到 CloudConvert 存储

3️⃣ 等待转换

GET /v2/jobs/{id}/wait — 轮询等待 convert 任务完成

4️⃣ 获取下载链接

从 export 任务结果中提取 RAR 文件的临时下载 URL

5️⃣ 下载 RAR

GET download_url — 下载 RAR 文件并保存到本地输出目录

FAQ

常见问题

点击展开查看解答

7-Zip 找不到怎么办？ ▼
默认路径 C:\Program Files\7-Zip\7z.exe，如果安装在其他位置，可通过 --seven-zip 参数指定路径。如果未安装，脚本会提示用户安装 7-Zip 或改用 Python zipfile 模块替代。
API Key 过期了怎么办？ ▼
CloudConvert API Key 有效期很长（到 2126 年），一般不会过期。如果返回 401 Unauthenticated，需要到 CloudConvert Dashboard → API → API Keys 重新生成，并更新 config.env 中的 Key。
免费额度用完了怎么办？ ▼
免费账号每天 25 次转换额度，每天零点重置。如果需要更多，可以升级 CloudConvert 付费套餐（$0.01/次），或者临时用 7z 格式替代（压缩率更高且无额度限制）。
大文件转换超时怎么办？ ▼
免费版单文件最大 1GB。如果上传/下载超时，通常是网络问题，建议重试。对于特别大的文件，可以考虑先在本地用 7-Zip 压缩以减小体积，再上传转换。

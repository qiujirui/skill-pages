Why

为什么需要内网穿透？

你在本地搭建的网站、页面、API 服务，默认只有同一局域网内才能访问。内网穿透把你本地的端口"搬"到公网上，生成一个 HTTPS 链接，任何人都能访问。

Commands

命令手册

6 条命令覆盖所有操作场景

🚀 一键启动 
tunnel.py start
启动 HTTP 服务器 + ngrok 隧道，自动验证，输出公网地址
📊 状态查询 
tunnel.py status
查看本地服务器和隧道运行状态、公网地址
🔄 重启服务 
tunnel.py restart
停止所有服务后重新启动，获取新公网地址
⏹️ 停止服务 
tunnel.py stop
停止 HTTP 服务器和 ngrok 隧道，清理进程
🖥️ 仅启服务器 
tunnel.py start-server
只启动本地 HTTP 服务器（端口 8099），不建隧道
🚇 仅建隧道 
tunnel.py start-tunnel
只启动 ngrok 隧道，前提是本地服务器已运行

Security

安全特性

内置安全 HTTP 服务器，防止敏感文件泄露

访问根目录为什么会显示 403？ ▼
这是安全设计。目录列表已被禁用，直接访问文件夹会返回 403。只有明确的 HTML 文件路径才能访问。
哪些文件类型可以访问？ ▼
允许：HTML, CSS, JS, JSON, PNG, JPG, GIF, SVG, MP4, PDF 等常见网页资源。
禁止：PY, EXE, ZIP, RAR, ENV 等代码、可执行文件、压缩包和配置文件。
能否防止路径遍历攻击？ ▼
可以。服务器已内置路径遍历防护，通过 ../ 等方式访问上级目录会被阻止。
FAQ

常见问题

首次访问 ngrok 地址为什么有个确认页？ ▼
这是 ngrok 免费版的正常行为，点击 Visit Site 即可进入目标页面。付费版没有这个确认页。
重启电脑后公网地址还能用吗？ ▼
不能。ngrok 免费版每次重启后地址会变。需要重新运行 tunnel.py start 获取新地址。
隧道断了怎么恢复？ ▼
运行 tunnel.py restart，会自动停止旧服务并重新启动。或分别运行 stop + start。
authtoken 失效了怎么办？ ▼
登录 dashboard.ngrok.com，在左侧菜单找到 Your Authtoken，复制新的 token，更新 config.env 中的 AUTHTOKEN 值。
可以固定公网域名吗？ ▼
ngrok 付费版支持固定域名。免费版只能用随机域名。如果需要固定域名，建议换用 Cloudflare Tunnel（免费固定域名）。

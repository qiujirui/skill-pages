1概述

Image Gen Skill 为企业员工提供基于内部 LLM 网关的 AI 图像生成服务，核心聚焦于 GPT-Image 2 图像生成。

核心能力：

  • 📋 自助申请：员工无需管理员介入，独立完成 API Key 申请
  • 🎨 图像生成：文生图、单图编辑、多图合成
  • 📊 额度管理：实时查询剩余 Token 额度

多平台支持：Python 脚本（首选）、PowerShell 脚本（Windows ACP 兼容）、Bash 脚本（macOS / Linux）

2核心能力

📝 文生图

根据文字描述生成图片。支持自定义尺寸、质量等级、文件名前缀。

python scripts/python/03_generate_image.py "中国山水画" 1088 1920 medium --prefix=landscape

✏️ 单图编辑

修改已有图片风格、内容。只需提供图片路径和编辑指令。

python scripts/python/04_edit_image.py D:\img.png "改成水墨风格" --quality high

🔀 多图合成

多张图片融合创作。支持 2~N 张图片合成一张海报。

python scripts/python/04_edit_image.py D:\a.png D:\b.png "合成一张海报" --width 1728 --height 3072

4质量选项

质量只影响处理耗时，不影响 token 消耗。

质量耗时说明
low~60 秒快速预览
medium~150 秒一般用途
high~360-600 秒最终交付，消耗 token 是标准的 15 倍

5铁律：不得擅自修改提示词

❌ 绝对禁止：将中文提示词翻译成英文、擅自添加修饰语、修改专有名词、将简洁描述"扩写"成一大段英文 prompt。
✅ 正确做法：用户说 "生成一张 GTA 风格的狗狗封面" → 直接原样传递 "生成一张 GTA 风格的狗狗封面"
唯一例外：当用户明确要求"帮我把这段话改成一个适合 AI 生成图片的 prompt"时，可以在用户确认后修改。

6实战经验与踩坑记录

❌ 问题：Bot 读取图片内容
图片编辑时 Bot 尝试"理解"图片内容后再生成编辑指令。
✅ 解决：Bot 只传递路径
Bot 只需要知道图片的文件路径，直接传给脚本即可。图片的"理解"由 API（gpt-image-2）完成。
❌ 问题：Windows ACP 模式阻止 Python 执行
ACP 模式下 PowerShell 无法启动外部子进程。
✅ 解决：自动切换 PowerShell 脚本
检测到 ACP 阻止后，自动切换到 scripts/powershell/ 目录下的 .ps1 脚本。
❌ 问题：Python 编码乱码
Python subprocess 调用脚本时中文提示词乱码。
✅ 解决：显式设置 UTF-8
result = subprocess.run( [sys.executable, "scripts/python/03_generate_image.py", "提示词"], capture_output=True, text=True, encoding='utf-8' )
❌ 问题：用户截图路径获取
用户插入截图时不知道图片保存路径。
✅ 解决：从 image_local_path 标签提取
系统会自动提供 <image_local_path> 标签，Bot 从中提取绝对路径直接传给脚本。

7错误速查

状态码含义处理
401Key 无效引导检查 Key 或重新申请
403Key 停用 / 额度不足显示剩余额度，联系管理员
408请求超时正常，图片生成需要 47~250 秒
413请求体过大缩小参考图后重试
500服务器错误联系管理员

8自助申请 API Key

首次使用需申请 API Key（格式：sk-self-xxx）

  1. 访问自助申请页面：http://10.225.9.142:31888/#/self-service
  2. 填写信息：姓名、企业邮箱、工号、企微部门
  3. 提交后获取 API Key（仅显示一次，请立即保存）
  4. 将 Key 发给 Bot 配置
